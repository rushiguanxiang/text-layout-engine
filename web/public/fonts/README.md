# web/public/fonts/README.md

## 字体来源

本演示站使用了以下开源字体：

### Noto Serif SC（中文主字体）
- 来源：Google Fonts
- 许可证：SIL Open Font License 1.1
- 用途：中文竖排、标点符号展示
- 字形范围：CJK 统一表意文字、CJK 符号和标点
- 下载地址：https://fonts.google.com/noto/specimen/Noto+Serif+SC

### Noto Sans（拉丁回退字体）
- 来源：Google Fonts
- 许可证：SIL Open Font License 1.1
- 用途：拉丁字母、数字、英文标点
- 字形范围：基本拉丁字母、拉丁补充-1
- 下载地址：https://fonts.google.com/noto/specimen/Noto+Sans

### Noto Sans JP（日文回退字体）
- 来源：Google Fonts
- 许可证：SIL Open Font License 1.1
- 用途：平假名、片假名、日文标点
- 字形范围：平假名、片假名、CJK 统一表意文字（日文风格）
- 下载地址：https://fonts.google.com/noto/specimen/Noto+Sans+JP

## 为什么选择 Noto 系列？

1. **统一的设计语言**：Noto 系列字体在不同书写系统之间保持一致的视觉风格
2. **广泛的字符覆盖**：覆盖 Unicode 中几乎所有的书写系统
3. **开放的许可证**：SIL OFL 允许免费使用、修改和分发
4. **Web 优化**：提供 woff2 格式，体积小，加载快

## 文件格式

我们使用 WOFF2（Web Open Font Format 2）格式，这是目前 Web 上最先进的字体格式：
- 比 WOFF 压缩率高 30-50%
- 所有现代浏览器均支持
- 加载速度快，减少带宽消耗
